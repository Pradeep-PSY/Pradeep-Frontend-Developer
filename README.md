# BrainStorm Assignment


## Tech Stack

**Client:** React, Redux, Chakra UI, 



## Run Locally

Clone the project

```bash
  git clone https://github.com/Pradeep-PSY/Company-Assignment/quiz-system/Frontend.git
```

Go to the project directory

```bash
  cd spacex
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
